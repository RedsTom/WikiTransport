# Initial application setup

Summary: First release of WikiTransport with a fully functional schematic transit map editor

Date: 2026-05-08

- Initialized SvelteKit project with TypeScript and Vite
- Added i18n with English and French translations (Paraglide-JS)
- Set up IndexedDB data layer with models and CRUD services (Dexie.js)
- Built a Material 3 UI component library (Button, Dialog, TextField, Select, etc.)
- Implemented editor state management with Svelte 5 runes
- Created SVG schematic plan view with grid, lines, stations, and anchors
- Added editor panels with property editors for stations, lines, and types
- Built project list page with create and delete dialogs
- Added full project editor page
