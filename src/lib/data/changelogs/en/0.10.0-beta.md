# Context menus everywhere

Summary: Right-click on anything to edit, add to, or remove from lines — plus a shared line picker component

Date: 2026-05-16

- Right-click on stations, lines, and types in the Overview tab for contextual actions (edit, delete, hide/show, add line)
- Right-click on the plan background to instantly add a station at that position
- Right-click on a line to instantly add an anchor point on the line
- Right-click on a station to remove it from a specific line
- Extracted `LinePicker` component used consistently in both toolbar and context menus
- New translations: `remove_from_line`, `edit_type`
