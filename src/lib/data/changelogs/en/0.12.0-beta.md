# Compact project export (.wtpc)

Summary: Export your projects in a compact GZIP-compressed format for smaller file sizes

Date: 2026-05-16

- New `.wtpc` (WikiTransport Project Compact) export format using GZIP compression
- Click the download button on any project card to choose between Normal (.wtp) and Compact (.wtpc)
- Compact files are significantly smaller thanks to GZIP compression
- Import auto-detects both formats — just select the file and it works
- New translations: `export_standard`, `export_compact`
