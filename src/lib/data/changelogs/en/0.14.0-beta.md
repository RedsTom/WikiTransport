# Tunnel-based rendering & corner radius

Summary: Complete line rendering overhaul with tunnel-aware offsets, segment ordering, and per-view corner rounding

Date: 2026-05-17

- Tunnel-based line rendering: lines now share paths through shared segments with proper offset computation
- Junction-aware offset propagation stops at stations for cleaner parallel runs
- Segment list in TunnelsTab with drag-to-reorder, hover highlight, collapsible groups
- Per-view corner radius on path angle nodes with slider and reset
- Corner properties panel (selectable from the segment list)
- Hover highlight for corners on the schematic
- Geometry utilities extracted to a dedicated module
