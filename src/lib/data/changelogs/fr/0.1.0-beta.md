# Configuration initiale de l'application

Summary: Première version de WikiTransport avec un éditeur de plans de transport fonctionnel

Date: 2026-05-08

- Mise en place du projet SvelteKit avec TypeScript et Vite
- Ajout de l'i18n avec traductions anglais et français (Paraglide-JS)
- Mise en place de la base de données IndexedDB avec modèles et services CRUD (Dexie.js)
- Création d'une bibliothèque de composants UI Material 3
- Gestion d'état de l'éditeur avec les runes Svelte 5
- Vue schématique SVG avec grille, lignes, stations et ancres
- Panneaux d'édition avec éditeurs de propriétés
- Page de liste des projets avec dialogues de création et suppression
- Page d'édition de projet complète
