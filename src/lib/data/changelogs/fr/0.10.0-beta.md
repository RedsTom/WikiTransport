# Des menus contextuels partout

Summary: Faites un clic droit sur n'importe quoi pour modifier, ajouter ou retirer des lignes — avec un sélecteur de ligne partagé

Date: 2026-05-16

- Clic droit sur les stations, lignes et types dans l'onglet Aperçu pour des actions contextuelles (modifier, supprimer, afficher/masquer, ajouter une ligne)
- Clic droit sur le fond du plan pour ajouter immédiatement une station à cet endroit
- Clic droit sur une ligne pour ajouter immédiatement un point d'ancrage sur la ligne
- Clic droit sur une station pour la retirer d'une ligne spécifique
- Composant `LinePicker` extrait et utilisé de façon cohérente dans la barre d'outils et les menus contextuels
- Nouvelles traductions : `remove_from_line`, `edit_type`
