# Export de projet compact (.wtpc)

Summary: Exportez vos projets dans un format compressé GZIP pour des fichiers plus légers

Date: 2026-05-16

- Nouveau format d'export `.wtpc` (WikiTransport Project Compact) utilisant la compression GZIP
- Cliquez sur le bouton de téléchargement d'un projet pour choisir entre Normal (.wtp) et Compact (.wtpc)
- Les fichiers compacts sont significativement plus légers grâce à la compression GZIP
- L'import détecte automatiquement les deux formats — il suffit de sélectionner le fichier
- Nouvelles traductions : `export_standard`, `export_compact`
