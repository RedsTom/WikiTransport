# Aperçus enrichis sur les réseaux

Summary: Vos liens affichent désormais un aperçu avec capture d'écran et description sur Discord, Twitter et autres plateformes

Date: 2026-05-16

- Ajout des meta tags Open Graph (og:) et Twitter Card pour les aperçus de liens enrichis
- En partageant un lien vers WikiTransport, les plateformes comme Discord affichent le nom du site, une capture d'écran et une description
- La description est traduite en anglais ou en français selon la langue du visiteur
