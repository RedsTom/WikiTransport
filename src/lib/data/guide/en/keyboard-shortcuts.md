## General

| Key     | Action                                |
| ------- | ------------------------------------- |
| `Space` | Recenter the canvas (fit all content) |
| `Esc`   | Cancel current mode / deselect        |
| `D`     | Deselect all                          |

## Creating Elements

| Key       | Action                                                          |
| --------- | --------------------------------------------------------------- |
| `S`       | Enter station placement mode (or add after a station on a line) |
| `Shift+S` | Add a new station before the selected station on a line         |
| `A`       | Enter anchor placement mode                                     |

## Selection

| Key                            | Action                                        |
| ------------------------------ | --------------------------------------------- |
| `Ctrl+click` or `Shift+click`  | Toggle multi-selection (stations and anchors) |
| `Shift+drag` (on empty canvas) | Rubber-band selection rectangle               |
| `Tab`                          | Next station on the selected line             |
| `Shift+Tab`                    | Previous station on the selected line         |

## Editing

| Key                           | Action                                        |
| ----------------------------- | --------------------------------------------- |
| `Delete` or `Backspace`       | Delete the selected element(s)                |
| Click + drag                  | Move a station or anchor                      |
| Hold `Ctrl` / `Shift` + click | Toggle in multi-select, then drag to move all |

## Toolbar

| Key   | Action                        |
| ----- | ----------------------------- |
| `S`   | Add station mode              |
| `A`   | Add anchor mode               |
| `D`   | Deselect everything           |
| `Esc` | Cancel current placement mode |

## Quick Reference

1. **Place stations:** `S`, click on canvas, type name, Enter
2. **Place anchors:** `A`, click on line
3. **Move things:** Click + drag
4. **Multi-select:** `Ctrl+click` or `Shift+click`
5. **Select rectangle:** `Shift+drag` on empty space
6. **Delete:** Select + `Delete`
7. **Done / cancel:** `Esc`
