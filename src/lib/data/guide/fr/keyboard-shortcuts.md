## Général

| Touche   | Action                                          |
| -------- | ----------------------------------------------- |
| `Espace` | Recentrer le canevas (afficher tout le contenu) |
| `Échap`  | Annuler le mode en cours / désélectionner       |
| `D`      | Tout désélectionner                             |

## Créer des éléments

| Touche  | Action                                                                 |
| ------- | ---------------------------------------------------------------------- |
| `S`     | Mode placement de station (ou ajouter après une station sur une ligne) |
| `Maj+S` | Ajouter une station avant celle sélectionnée sur une ligne             |
| `A`     | Mode placement d'ancre                                                 |

## Sélection

| Touche                       | Action                                                     |
| ---------------------------- | ---------------------------------------------------------- |
| `Ctrl+clic` ou `Maj+clic`    | Activer/désactiver la multi-sélection (stations et ancres) |
| `Maj+glisser` (canevas vide) | Rectangle de sélection                                     |
| `Tab`                        | Station suivante sur la ligne sélectionnée                 |
| `Maj+Tab`                    | Station précédente sur la ligne sélectionnée               |

## Édition

| Touche                      | Action                                                      |
| --------------------------- | ----------------------------------------------------------- |
| `Suppr` ou `Retour arrière` | Supprimer le(s) élément(s) sélectionné(s)                   |
| Clic + glisser              | Déplacer une station ou une ancre                           |
| `Ctrl` / `Maj` + clic       | Activer la multi-sélection, puis glisser pour tout déplacer |

## Barre d'outils

| Touche  | Action                                |
| ------- | ------------------------------------- |
| `S`     | Mode ajouter une station              |
| `A`     | Mode ajouter une ancre                |
| `D`     | Tout désélectionner                   |
| `Échap` | Annuler le mode de placement en cours |

## Référence rapide

1. **Placer des stations :** `S`, clic sur le canevas, taper le nom, Entrée
2. **Placer des ancres :** `A`, clic sur une ligne
3. **Déplacer :** Clic + glisser
4. **Multi-sélection :** `Ctrl+clic` ou `Maj+clic`
5. **Sélection rectangle :** `Maj+glisser` sur l'espace vide
6. **Supprimer :** Sélectionner + `Suppr`
7. **Terminé / annuler :** `Échap`
