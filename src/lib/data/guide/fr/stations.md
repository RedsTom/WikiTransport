## Ajouter une station

**Méthode 1 — Barre d'outils :** Cliquez sur <guide-btn icon="add_location">Ajouter une station</guide-btn> dans la barre d'outils en bas (ou appuyez sur `S`). Le curseur devient une croix. Cliquez n'importe où sur le canevas pour placer la station, puis saisissez son nom dans la boîte de dialogue.

**Méthode 2 — Clic droit :** Faites un clic droit sur l'espace vide du canevas et choisissez <guide-btn icon="add_location">Ajouter une station</guide-btn>. La station sera placée à l'endroit où vous avez cliqué.

## Sélectionner une station

Cliquez simplement sur une station pour la sélectionner. Une station sélectionnée a un **contour noir épais**. Le panneau droit affiche ses propriétés.

## Déplacer une station

Cliquez et glissez une station pour la déplacer. Les lignes connectées suivent automatiquement.

## Multi-sélectionner des stations

Vous pouvez travailler avec plusieurs stations à la fois :

- **Ctrl+clic** ou **Maj+clic** sur une station pour l'ajouter à (ou la retirer de) la sélection
- **Maj+glisser** sur le canevas vide dessine un **rectangle de sélection** — tout ce qui se trouve à l'intérieur est sélectionné
- Une fois plusieurs stations sélectionnées, **glissez n'importe laquelle** pour toutes les déplacer ensemble. La barre d'outils affiche un compteur comme "3 Stations".
- Appuyez sur `Suppr` pour supprimer toutes les stations sélectionnées à la fois

## Propriétés d'une station

Sélectionnez une station pour modifier ses propriétés dans le panneau droit :

- **Nom** et **sous-titre** — le texte de l'étiquette
- **Position** (X, Y) — ajustez finement la position
- **Direction de l'étiquette** — choisissez où le nom apparaît : N, NE, E, SE, S, SO, O, NO
- **Distance de l'étiquette** — l'éloignement du libellé par rapport à la station (Dx, Dy)
- **Alignement du sous-titre** — gauche, centre ou droite

Tous les paramètres d'étiquette peuvent aussi être ajustés par vue.

## Badges de correspondance

En mode **vue**, les lignes masquées peuvent apparaître comme **badges de correspondance** sur les stations qu'elles desservent. Utile pour indiquer "vous pouvez correspondre ici" sans encombrer le plan.

Pour configurer :

1. Passez à une vue non globale
2. Sélectionnez une station desservie par une ligne masquée
3. Dans le panneau droit, sous **Affichage des correspondances**, choisissez :
   - **Par rapport à la station** — badges positionnés près du point de la station
   - **À côté du texte** — badges alignés à côté du nom
   - **Empilé avec le texte** — badges empilés avec le nom
4. Ajustez la **Position des correspondances** parmi 8 directions
5. Activez/désactivez les lignes individuellement

## Supprimer une station

Sélectionnez une station et appuyez sur `Suppr` ou `Retour arrière`, ou faites un clic droit et choisissez <guide-btn icon="delete">Supprimer la station</guide-btn>. Vous pouvez aussi utiliser le bouton **Supprimer** dans le panneau des propriétés.
